# SEO: Карта URL, H1, Title и Meta Description

Дата: 2026-04-05

## 1. Задача этого файла

Этот документ — рабочая карта для следующего этапа SEO-расширения проекта.

Здесь собраны:

- рекомендуемые URL
- H1
- Title
- Meta Description
- роль каждой страницы в SEO-структуре

Документ рассчитан на русскоязычную версию сайта.

### Зафиксированное решение на этапе 1

Для этого документа принимается рабочая рамка:

- рынок: русскоязычный
- язык SEO-страниц: русский
- текущая карта URL относится к `RU-first` стратегии

Важно:

- это не финальное решение “навсегда”
- это решение для первого SEO-цикла, чтобы не строить структуру сразу на два рынка

## 2. Базовые правила для новых страниц

### URL

- только lowercase
- слова через дефис
- коротко, без лишних предлогов
- без дат и лишних хвостов

### Title

- желательно в диапазоне 50–60 символов
- сначала смысл и ключевой интент
- бренд в конце только если это не ломает длину

### Meta Description

- желательно около 130–160 символов
- без переспама
- с понятной ценностью страницы

### H1

- один H1 на страницу
- ближе к интенту пользователя, а не к “брендовому слогану”

## 7. Внутренняя логика перелинковки

### Главная

Ссылки с главной:

- на все 3 услуги
- на все 3 кейса
- на 3 статьи

### Страницы услуг

Каждая страница услуги должна ссылаться:

- на 1–2 релевантных кейса
- на 1–2 статьи
- на контакт / разбор сайта

### Кейсы

Каждый кейс должен ссылаться:

- на соответствующую услугу
- на контактную страницу

### Статьи

Каждая статья должна ссылаться:

- на релевантную услугу
- на релевантный кейс
- на CTA-раздел или contact page

## 8. Приоритет публикации

После первого SERP-review приоритет выглядит так:

1. Главная — довести до конца
2. Подготовить многостраничную структуру
3. `audit-i-redizain-saita-na-konversiyu`
4. `kova-technology-ecommerce`
5. `pochemu-sait-ne-konvertiruet`
6. `razrabotka-b2b-saitov-i-veb-interfeisov`
7. supporting case для B2B
8. `chto-dolzhno-byt-na-b2b-sajte`
9. Дополнительный SaaS long-tail research
10. Только потом решать судьбу `razrabotka-saitov-dlya-saas`
11. `skolko-stoit-razrabotka-saita-dlya-saas-i-b2b`

Логика:

- сначала публикуется кластер `Audit / Conversion`
- затем второй кластер `B2B`
- SaaS не выпускается в первую волну по одному head-term предположению

## 9. Практическое замечание

Все titles и descriptions в этом файле — стартовые рабочие версии.

После публикации их стоит корректировать по данным:

- Search Console impressions
- CTR
- реальные поисковые формулировки
- сниппеты конкурентов

То есть это не “финал навсегда”, а правильная отправная точка.
